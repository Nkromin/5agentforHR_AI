IT SECURITY POLICY

Section 1: Password Policy
- Passwords must be at least 14 characters long
- Must include uppercase letters, lowercase letters, numbers, and special characters
- Passwords must be changed every 90 days
- Cannot reuse the last 10 passwords
- Do not share passwords with anyone including IT staff
- Use company-approved password manager for secure storage
- Enable multi-factor authentication on all accounts

Section 2: Data Security
- All company data must be stored on approved systems only
- Do not use personal cloud storage for company files
- Encrypt sensitive files before sharing externally
- Report any data breaches immediately to IT Security team
- Classify data according to sensitivity levels
- Delete sensitive data securely when no longer needed

Section 3: Device Security
- Lock your computer when leaving your desk using Windows+L or Cmd+L
- Enable full disk encryption on all devices
- Install security updates within 48 hours of release
- Do not connect unauthorized USB devices to company computers
- Do not install unapproved software on work devices
- Report lost or stolen devices immediately to IT

Section 4: Network Security
- Do not use public WiFi for accessing company systems or data
- Always connect through company VPN when working remotely
- Do not share WiFi passwords with non-employees
- Report suspicious network activity to IT Security
- Do not attempt to bypass network security controls

Section 5: Email Security
- Do not click suspicious links or attachments
- Report phishing emails to security@company.com immediately
- Use company email for business purposes only
- Enable two-factor authentication on email accounts
- Verify sender identity before sharing sensitive information
